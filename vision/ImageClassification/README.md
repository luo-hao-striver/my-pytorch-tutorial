# 图像分类
# 1.数据集种类
- 本次图像分类使用的数据集是FashionMNIST
# 2.数据集获取方式
- 方式一：可以到pytorch官网查到FashionMNIST数据集，自行下载，放到dataset文件夹下即可
- 方式二（recommend）：直接运行train.py，程序会自行下载数据集

# 3.train the model
run the train.py
```Python
python train.py
```

# 4.test the model
run the test.py
```Python
python test.py
```
# 5.models.py
- 这是图像分类构建的分类模型
- 大家可以把自己的模型放入此处并调用
